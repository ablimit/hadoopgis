The package xxl.core.collections.queues provides miscellaneous implementations of the queue datastructure. It contains a bounded queue ({@link xxl.core.collections.queues.BoundedQueue}), queues using inserting strategies like LIFO ( {@link xxl.core.collections.queues.StackQueue}) and FIFO ( {@link xxl.core.collections.queues.ListQueue}), queues using special data structures for internal storing (for instance {@link xxl.core.collections.queues.ArrayQueue}), heaps ( {@link xxl.core.collections.queues.Heap}) and even a threadsafe queue ( {@link xxl.core.collections.queues.ThreadsafeQueue}).
