The package xxl.core.cursors.wrappers provides miscellaneous implementations of wrapper classes that make an object representing an arbitrary form of iteration via the {@link xxl.core.cursors.Cursor cursor} interface. Wrappers are provided for:
  • {@link java.util.Enumeration enumerations}
  • {@link java.util.Iterator iterators}
  • {@link xxl.core.collections.queues.Queue queues}
